# Painel de Faltas — Televendas

Painel operacional para gestão de faltas, conferência, estoque e recebimento.

## 🔗 Acesso
O painel está disponível em:  
**https://SEU-USUARIO.github.io/painel-faltas**

## 📋 Como usar

1. Abra o link acima no navegador (PC ou celular)
2. Os dados do mês atual carregam automaticamente do servidor
3. Para atualizar os dados, faça upload dos arquivos:
   - **PDF Faltas** — Relatório de Faltas Televendas (Manager of Decision)
   - **PDF Conferência** — Relatório de Conferência (Manager of Decision)
   - **Estoque (Excel)** — Arquivo de estoque extraído do sistema
   - **Recebimento (Excel)** — Planilha de recebimento (Col C: Data, D: Código, G: Qtd)

## 🗄️ Banco de dados
Os dados ficam salvos no Supabase (PostgreSQL na nuvem).  
Qualquer pessoa com o link acessa os mesmos dados em tempo real.

## 📱 Compatível com
- Desktop (Chrome, Firefox, Safari, Edge)
- iPhone e Android (Chrome, Safari)
